​	然后今天，还是在洛谷上，刷了几道入门题，熟悉一下Go的书写，虽然还是，一些简单的操作，不过，对比昨天，今天码入的速度要快上不少，也更加熟悉了Go的语法，不过那些简单题也是有收获的，就是Go里面是不会自动数据类型转换的，要自己手动强转，比如 

~~~
var a , b int
fmt.Scan(&a,&b)
ans = float64(a) / float64(b)
fmt.Printf("%.4f",ans)
~~~

值得一提的是这个括号是给变量的，不是给数据类型的！
